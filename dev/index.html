SmoreExamples

Documentation for SmoreExamples.

A bundle of runnable Pluto notebooks demonstrating the SmoreVerse modeling pipeline.

Available examples

Call list_examples to see every bundled notebook and its title:

list_examples()
4 example notebooks (run with `run_example("<file>")`):
  cm_posterior_pipeline.jl       CM Posterior from Real Data (SmoreFit)
  logistic_growth_pipeline.jl    The SmoreVerse Pipeline: Logistic Growth Tutorial
  replicate_runs_custom_loss.jl  Custom CM Data: Raw Replicate Runs with a Custom Loss
  single_obs_custom_data.jl      Custom CM Data: Single Observation Per Parameter Set

Launch one with, e.g., `run_example("cm_posterior_pipeline.jl")`.

Open any of them in Pluto with run_example:

using SmoreExamples
run_example("cm_posterior_pipeline.jl")

The notebooks are copied to a temporary directory (pass dir = "/your/path" to keep edits) and opened with the SmoreExamples project activated. Omit the name to open logistic_growth_pipeline.jl.

SmoreExamples.ExampleInfo — Type

A single bundled example notebook: its file name (pass this to run_example) and the title taken from the notebook's first heading.

SmoreExamples.list_examples — Method
list_examples() -> ExampleList

List the example notebooks bundled with SmoreExamples, with the title of each.

Pass any listed file to run_example to open it in Pluto.

SmoreExamples.run_example — Function
run_example(name = "logistic_growth_pipeline.jl"; dir = nothing)

Open the bundled example notebook name in Pluto with the SmoreExamples project activated. See list_examples for the available names.

The notebooks are copied to dir (a temporary directory by default; pass a path to keep your edits) before being opened.
